# Toggle — Bundle & Save

A sleek, interactive UI for selecting product bundles with dynamic pricing and options. Built with vanilla HTML, CSS, and JavaScript.

## Features

- **Bundle Selection**: Choose between 1, 2, or 3-pair bundles
- **Dynamic Pricing**: Real-time price updates based on selected bundle
- **Size & Color Options**: Select size (S, M, L) and color (Black, White) for each pair
- **Responsive Design**: Clean, centered layout with smooth animations
- **Popular Badge**: Highlights the most popular bundle option (40% discount)

## Project Structure

```
toggle/
├── index.html      # Main HTML markup
├── style.css       # Styling with transitions
├── script.js       # Bundle selection logic
└── README.md       # This file
```

## Usage

1. **Open in Browser**: Simply open `index.html` in any modern web browser.
2. **Select Bundle**: Click a radio button to choose between 1, 2, or 3-pair options.
3. **Customize**: Select size and color for each pair in the selected bundle.
4. **View Total**: The total price updates automatically based on selection.
5. **Add to Cart**: Click the "Add to Cart" button to proceed.

## Technologies

- **HTML5** — Semantic markup for bundle options and form elements
- **CSS3** — Flexbox layout, transitions, and responsive design
- **JavaScript (Vanilla)** — Dynamic bundle toggle and price updates

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge).

## Notes

- Free 2-day shipping is included
- Pricing is displayed in Danish Krone (DKK)
- The "Most Popular" badge highlights the 2-pair bundle with the best value
